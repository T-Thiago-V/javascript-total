Evento onclick:—Este atributo responde al evento click. El evento se ejecuta cuando el usuario hace clic con el botón izquierdo dek ratón

Evento onmousedown:—Este atributo responde al evento click. El evento se ejecuta cuando el usuario de este evento se desencadena cuando el usuario pulsa el botón izquierdo o el derecho

Evento onmouseup:—Este atributo responde al evento mouseenter. Este evento se desencadena cuando el ratón se introduce en el área ocupada por el elemento.

Evento onmouseenter:Este atributo responde al evento mouseleave. Este evento se desencadena cuando el ratón abandona el área ocupada por el elemento.

Evento onmouseleave:Este atributo responde al evento mouseleave. Este evento se desencadena cuando el ratón abandona el área ocupada por el elemento.

Evento onmouseover:Este atributo responde al evento mouseover. Este evento se desencadena cuando el ratón se mueve sobre el elemento o cualquiera de sus elementos hijos.

Evento onmouseout:—Este atributo responde al evento mouseout. El evento se desencadena cuando el ratón abandona el área ocupada por el elemento o cualquiera de sus elementos hijos.

Evento onmousemove:Este atributo responde al evento mousemove. Este evento se desencadena cada vez que el ratón se encuentra sobre el elemento y se mueve.

Evento onwheel:—Este atributo responde al evento wheel. Este evento se desencadena cada vez que se hace girar la rueda del ratón.
